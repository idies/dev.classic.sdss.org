News and updates

DR3 available

September 27, 2004: Data Release 3 (DR3) is available, with more sky coverage and better reductions. We strongly encourage you to use DR3 data.

Old "news"

DR1 White Dwarf Catalog available

February 27, 2004: The DR1 Spectroscopic White Dwarf Catalog by Kleinman et al. is available from the value added catalogs page.

Catalog Archive Server released

June 11, 2003: Catalog Archive Server for DR1 available for public access. The Catalog Archive Server offers efficient search tools for querying the imaging and spectro catalogs from DR1.

DR1 "beta" available

The present data release contains the "beta" processing of Data Release 1. Read About DR1 for details.
