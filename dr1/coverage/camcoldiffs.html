List of "missing" fields that are not meant to be in DR1

Usually, the range of field numbers within a run is identical for all six camera columns. But this is not always the case. If you have discovered that some fields are missing for some camera columns of some runs, check this list for the exceptions. The "symptom" is a mis-match of startfield and/or endfield in the chunk file across 1 or more columns.

Note: If any of the terms on this page do not mean anything to you, you can safely ignore this page.
