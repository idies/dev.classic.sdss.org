Where to start

About the data

  • About DR2 gives an overview of what DR2 contains, and a summary of the improvements in DR2 compared to earlier data releases. The DR2 paper (Abazajian et al. 2004, AJ, submitted; astro-ph/0403325)contains the details, and the respective sections of this web site have been updated accordingly. The most important change is that the DR1 contained the "beta" processing of survey data. DR2 has bug-free photometry and vastly improved spectrophotometry.
  • The Early Data Release (EDR) paper (with a journal subscription, access the EDR paper at the AJ web site) is the most comprehensive printed description of SDSS data. Much of the information on this web site is based on the EDR paper. It should be considered essential reading to do science with SDSS data.
  • However, some of the information in the EDR paper has been superseded by information on this web site. There is a description of changes from the Early Data Release to Data Release 1 (DR1) at the DR1 site (also see the DR1 paper Abazajian et al. 2003, AJ 126, 2081 (AJ site, subscription required); also at astro-ph/0305492).

  • These pages help to avoid the most common pitfalls in retrieving and using SDSS data:

    • When you use the photometric object catalogs, you must check the photometric object flags to obtain meaningful results.
    • When you use the spectroscopic catalogs, the photometric flag checking has been done as part of target selection. However, you must check the redshift status flag to exclude unsecure identifications and redshifts.
    • When you use the SQL database of the Catalog Archive Server (CAS), please read the advice on optimising CAS queries to avoid unnecessarily slow queries.

    Other caveats are detailed on the individual pages for each of our data products.
